Half Edge Mesh

Author: Jordan Smith (jordan.philip.smith@gmail.com)
Date: May 3, 2024

Half edge mesh to represent 2-manifolds. This is a modification of the quad edge data structure. The 2-manifold surface mesh is made up of vertices, facets, and half edges.

Vertex:

The vertex contains the position. The vertex points to one half edge in its CCW (counterclockwise) half edge ring. The valence is the number of half edges (i.e. vertex neighbors).

Facet:

A facet is a simple face loop. The facet points to one half edge in its CCW half edge ring. The valence is the number of half edges (i.e. vertices).

Edge and Half Edge:

Every edge is represented by two opposite direction half edge pairs. The half edge from V1->V2 has an opposite partner pair from V2->V1. An edge and its two half edges are allocated and deallocated as a unit using HEAllocate() (one of the two half edges is returned).

Each half edge points to a vertex, a facet, its opposite partner half edge, its edge, the next half edge around the vertex, and the next half edge around the facet.

Half edges on the boundary of the mesh point to the invalid facet (kFAInvalid). There may be multiple connected components, so it is not possible to represent the boundary area with a single simple facet. The boundary half edges are members of a set, hence the boundary can be iterated over.

Example

Run

tools/delaunay_example.bash

to generate random points, the Delaunay triangulation, and the convex hull. This runs the following tools.

random_points_cli

Generates N random points within a 2D bounding box.

tools:random_points_cli

delaunay_triangulator_cli

Generates the Delaunay triangulation of a set of 2D points represented by the vertex information only of a .obj file.

tools:delaunay_triangulator_cli

convex_hull_2d_cli

Generates the 2D convex hull of a set of 2D points represented by the vertex information only of a .obj file.

tools:convex_hull_2d_cli

obj_to_svg_cli

Converts the edges of a polygon mesh stored as a .obj file to a .svg file.

tools:obj_to_svg_cli

Results

Delaunay Triangulation

The random points are the vertices of the Delaunay triangulation. A property of the Delaunay tringulation is that the circumcircle of each Delaunay triangle is empty of all other vertices.

Convex Hull

The boundary of the Delaunay triangulation is the convex hull polygon of the points. Here is the convex hull of the random points.
